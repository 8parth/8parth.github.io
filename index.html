ADBM Smooth Video Streaming Sevice

We achieved our aim to reduce the waiting time as much low as possible by analyzing bandwidth of users dynamically on run-time and provide appropriate resolution and quality video which adapts user's bandwidth.

The application is built using Java EE and various aws services. We used AWS elastic transcoder and HLS player to achieve this.

Show Feeds Get information of more than 250 TV shows.

This application crawls data from http://showrss.info to get xml feed of clicked TV show with mechanize gem, and parses the xml feed with feedjira gem to show the details of show and torrent download links of latest released episodes. Feeds get updated by showrss and hence user can get torrent download link of latest episodes.
